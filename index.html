Gyan's API Documentation

Base URL: https://api.gyanl.com

Endpoints

Get Random Fact

Returns a random fact about Gyan.

Endpoint: GET /fact

Response:

  • Returns a JSON object containing a random fact and the total number of available facts

Example Request:

curl https://api.gyanl.com/fact

Example Response:

{
  "fact": "Gyan is a software engineer who loves building things",
  "total_facts": 5
}

Error Response:

{
  "error": "Internal Server Error",
  "message": "Something went wrong"
}

Generate Acronym Expansions

Generates creative acronym expansions for a given prompt.

Endpoint: GET /acronyms/:prompt

Parameters:

  • prompt (path parameter): The acronym to generate expansions for

Response:

  • Returns a JSON array of strings, where each string is an HTML-formatted expansion
  • Each expansion is a complete sentence starting with a letter from the acronym

Example Request:

curl https://api.gyanl.com/acronyms/API

Example Response:

[
  "Amazing Programming Interface",
  "Always Providing Innovation",
  "Advanced Problem Integration",
  "Automated Process Intelligence",
  "Agile Project Implementation"
]

Error Response:

{
  "error": "An error occurred while processing your request",
  "details": "Error message details"
}

Rate Limits

  • Standard rate limits apply
  • Please be mindful of API usage

Authentication

  • No authentication required
  • Public API endpoint

CORS

  • CORS is enabled
  • All origins are allowed (Access-Control-Allow-Origin: *)

Notes

  • The API uses OpenAI's GPT-3.5 Turbo Instruct model
  • Each request generates 10 different creative expansions
  • Responses are formatted in HTML for easy integration
